# Facturación

**Arquos Facturame** te permite generar tu CFDI de manera rápida y sencilla. A continuación, te explicamos cómo utilizar este módulo.

## Selección de organismo *

Para comenzar, debes seleccionar el organismo al que pertenecen los tickets que deseas facturar.

**Imagen**

En caso de que no aparezca este selector de organismos, no tienes que preocuparte, sigue con los siguientes pasos.

## Agregar RFC *

A continuación, debes ingresar el RFC al que se hará el CFDI. Es muy importante que el RFC sea correcto, ya que este dato es fundamental para la elaboración del CFDI.

***Imagen**

## Datos Fiscales *

En la siguiente sección, deberás ingresar tus datos fiscales. Estos datos son necesarios para la elaboración del CFDI y deben ser correctos. Asegúrate de ingresar tu razón social, código postal, régimen fiscal y uso de CFDI.

**Imagen**

## Datos de contacto *

En esta sección, deberás ingresar tus datos de contacto. Estos datos son importantes para que puedas recibir tu CFDI por correo electrónico y para que puedas ser contactado en caso de ser necesario. Asegúrate de ingresar un correo electrónico válido y un número de teléfono.

**Imagen**

## Captura de folios *

En la siguiente sección, deberás ingresar el o los folios de los tickets que deseas facturar. Puedes ingresar varios folios, uno a la vez. Deberás ingresar al menos un folio para poder generar tu CFDI.

**Imagen**

Asegúrate de ingresar los folios correctamente, ya que estos datos son necesarios para la elaboración del CFDI, si tienes dudas de donde encontrar los folios, puedes precionar el botón de ayuda que se encuentra al lado del campo de texto, este te mostrará una imagen de ejemplo de donde encontrar los folios en tus tickets.

**Imagen**

Al ingresar un folio, el sistema verificará si el folio es válido y si aún no ha sido facturado. Si el folio es válido y esta disponible para facturación, se mostrarán los datos del folio, incluyendo el monto total y la fecha de emisión del ticket. 

**Imagen**

Si deseas eliminar un folio ingresado, puedes hacerlo presionando el botón de eliminar que se encuentra al lado del folio.

**Imagen**

Si el folio ya ha sido facturado, se mostrará un mensaje indicando que el folio ya ha sido utilizado.

**Imagen**
--
Una vez que hayas ingresado todos los folios que deseas facturar, presiona el botón de "Siguiente" para continuar con el proceso de generación del CFDI.

## Selección de forma de pago

En esta sección, deberás seleccionar la forma de pago que utilizaste para realizar el pago de los tickets. En caso de que tus tickets los hayas pagado con formas de pago diferentes entre sí, se recomienda que generes un CFDI por cada forma de pago utilizada.

**Imagen**

## Verificación de datos

**Imagen**

### Revisión de datos fiscales

En esta sección, podrás revisar los datos fiscales que ingresaste anteriormente. Asegúrate de que todos los datos sean correctos, ya que estos datos serán utilizados para la generación del CFDI. En caso de que necesites realizar alguna modificación, puedes hacerlo presionando el botón de "Atrás".

### Botón atrás

Este botón te permitirá regresar a la sección anterior para realizar modificaciones en los datos ingresados.

### Botón de vista previa

Este botón te permitirá ver una vista previa del CFDI que se generará.

**Imagen**

### Botón de facturación

Este botón te permitirá generar el CFDI. Al presionarlo, se iniciará el proceso de generación del CFDI (durante este momento no deberás dar clic en ningún otro lado) y se enviará un correo electrónico con el CFDI generado (XML y PDF) a la dirección de correo electrónico que ingresaste anteriormente.

## Resultados del proceso

### Correo electrónico

Una vez que el CFDI ha sido generado, recibirás un correo electrónico con el CFDI en formato XML y PDF adjunto. Asegúrate de revisar tu bandeja de entrada y tu carpeta de spam.

**Imagen**

### Redirección a consultar CFDI

Una vez que hayas generado tu CFDI, serás redirigido al módulo de [Consultar CFDI](/docs/consultar-cfdi), donde podrás ver el CFDI y descargarlo si es necesario.

**Imagen**
