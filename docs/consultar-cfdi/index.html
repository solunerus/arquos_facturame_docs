Consultar CFDI

Selección de organismo​

Para comenzar, debes seleccionar el organismo al que pertenecen los tickets que deseas facturar.

Una vez seleccionado, aparecerá en la parte superior la información del organismo.

En caso de que no aparezca este selector de organismos, no tienes que preocuparte, sigue con los siguientes pasos.

Búsqueda de folio​

Para consultar el estado de un CFDI, debes ingresar el folio de solicitud o el folio de algún ticket que hayas facturado anteriormente. Este folio es de vital importancia para identificar el estado de tu CFDI en el sistema.

A un costado del campo de texto, encontrarás un botón de ayuda que te mostrará una imagen de ejemplo de dónde encontrar el folio en tus tickets o en el correo electrónico que recibiste al solicitar el CFDI.

Resultados de la consulta​

Una vez que hayas ingresado el folio, presiona el botón de "Consultar" para obtener los resultados. El sistema te mostrará el estado del CFDI.

En caso de error, el sistema te mostrará un mensaje con el motivo.

Estados posibles​

  • Generado: El CFDI ha sido generado exitosamente y está disponible para descarga.
  • En proceso: El CFDI está siendo procesado y aún no está disponible.
  • Rechazado: El CFDI no pudo ser generado debido a algún error o falta de información.
