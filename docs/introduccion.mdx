# Introducción

**Arquos Facturame** es una herramienta diseñada para facilitar la gestión de facturación electrónica en ArquosCRM. Con esta herramienta, podrás consultar, generar o solicitar tu CFDI de manera sencilla y rápida, todos desde la comodidad de tu navegador.

<ImageDiv src="/img/arquos_facturame/introduccion.png" alt="Arquos Facturame" />

## ¿Cómo empezar?

Como se puede observar en la imagen, **Arquos Facturame** consta de tres módulos principales:

- [**Facturación**](/docs/facturacion): Aquí podrás generar tu CFDI de manera rápida y sencilla, los archivos de tu factura llegarán a tu correo electrónico y podrás descargarlos directamente desde la plataforma.
- [**Solicitar CFDI**](/docs/solicitar-cfdi): Si no te es posible generar tu CFDI, puedes solicitarlo a través de este módulo. Solo necesitas ingresar los datos necesarios y el sistema se encargará del resto. A diferencia del módulo de facturación, aquí no se generará tu CFDI automáticamente, sino que se enviará una solicitud para que el CFDI sea generado por el sistema ArquosCRM. Se te brindará un folio que podrás utilizar para consultar el estado de tu solicitud.
- [**Consultar CFDI**](/docs/consultar-cfdi): Este módulo te permite consultar el estado de tus CFDI generados o solicitados. Podrás ver el estado de tu CFDI, si ha sido generado, si está en proceso o si ha sido rechazado. También podrás descargar tus CFDI desde aquí.

## Requisitos

Para utilizar **Arquos Facturame**, no necesitas crear una cuenta, basta con tener tus datos fiscales y los folios de los tickets que deseas facturar.
